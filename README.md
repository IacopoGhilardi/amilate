# Amilate 🕐

> **Am I Late?** — Never be late again.

Amilate is an iOS app for managing your appointments intelligently. Add where you need to go and when, and the app automatically calculates when you should leave based on real-time traffic — notifying you at exactly the right moment.

---

## Features

- **Manage your appointments** — title, date, time and destination in a few taps
- **Automatic departure calculation** — estimates travel time based on current traffic
- **Smart notifications** — alerts you exactly when it's time to leave
- **Real-time status** — always know if you're on time, should leave now, or running late (and by how much)
- **Saved destinations** — save your frequent places to add them even faster
